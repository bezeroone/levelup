# 🏫 초등 LEVEL UP+ 탐험대 (Elementary Level Up+ Expedition)

초등학생과 교사를 위한 디지털 교수·학습 자료 통합 플랫폼입니다. 재미있는 게임과 학습 도구를 통해 아이들의 독서력과 학습 흥미를 높일 수 있도록 설계되었습니다.

## 🌟 주요 기능 (Features)

### 1. 메인 탐험 지도 (`home/`)
- 슈퍼 마리오 스타일의 직관적인 섬 지도 인터페이스
- 애니메이션 효과와 함께 각 학습 구역으로 이동
- 🦖 알을 클릭하면 태어나는 귀여운 아기 공룡 커서!

### 2. 수업 플레이존 (`1/`)
신나는 게임을 통해 배우는 놀이 공간! (놀이동산 컨셉)
- **🧱 책 쌓기 테트리스**: 블록을 쌓아 독서 단어를 완성하는 테트리스 게임
- **🛸 반짝반짝 우주 타자**: 우주 배경에서 즐기는 타자 연습
- **🎓 한자 짝꿍 찾기**: 같은 한자와 뜻을 연결하는 메모리 게임
- **🐣 독서펫 키우기**: 독서 활동 기록으로 펫을 알에서 부화시키는 게임
- **🏫 학교 생활 지킴이**: O/X 퀴즈로 배우는 올바른 생활 습관

### 3. 성취기준 도서검색맵 (`2/`)
- 학년별/과목별 성취기준과 연계된 도서를 쉽게 찾을 수 있는 검색 도구

### 4. 바로쓰는 독서학습지도안 (`3/`)
- 교사를 위한 AI 기반 맞춤형 독서 수업 지도안 설계 도우미

### 5. 독서력 향상 활동지 (`4/`)
- 게임 속 도서관 같은 몰입감 있는 디자인
- **SQ3R, KWL, 인물 분석** 등 다양한 독서 전중후 활동지 제공

## 🛠️ 기술 스택 (Tech Stack)
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS)
- **Design**: Google Fonts (Jua, Noto Sans KR, Yeonsung), CSS Animations, Pixel Art Assets
- **Compatibility**: Chrome, Edge 등 모던 브라우저 권장

## 🚀 실행 방법 (How to Run)
1. 이 저장소를 다운로드하거나 클론(Clone)합니다.
2. 프로젝트 폴더 내 `home` 폴더로 이동합니다.
3. `index.html` 파일을 더블 클릭하여 웹 브라우저에서 실행합니다.
   - **시작 경로**: `.../digital_teamwork/home/index.html`

## 📂 폴더 구조 (Folder Structure)
```
digital_teamwork/
├── home/          # 메인 홈페이지 (index.html) 및 공통 스타일/스크립트

├── 2/             # 성취기준 도서검색맵
├── 3/             # 바로쓰는 독서학습지도안

```
